## 🔑 마지막 단계: API 키 설정

### 1. Gemini API 키 발급
1. **Google AI Studio 접속**: https://aistudio.google.com/
2. **Get API Key** 클릭
3. **Create API Key** 선택
4. **키 복사** (sk-으로 시작하지 않음, 일반 문자열)

### 2. API 키 설정
```bash
# .env.local 파일에서 PLACEHOLDER_API_KEY를 실제 키로 교체
GEMINI_API_KEY=실제발급받은키여기입력
```

### 3. 즉시 테스트
1. **브라우저에서**: http://localhost:5174
2. **채팅창에 입력**: "안녕하세요"
3. **예상 결과**: 1-3초 내 한국어 응답

## 📊 비용 최적화 기능

### 자동 사용량 관리
